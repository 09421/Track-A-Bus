\subsection{Use Case realiseringer}

Dette afsnit beskriver, hvordan de fastsatte Use Cases i kravspecifikationen er realiseret. Dette indeholder initialisering, process og afsluttelse. Samtlige Use Cases startes i præsentations laget, men interagerer på forskellige måder med systemet. Ikke Use Case relevante processer (funktionalitet der ikke er strengt use case relateret) beskrives i \textit{afsnit 8: Implementerings View}, for den relevante komponent, Use Casen udføres på.
