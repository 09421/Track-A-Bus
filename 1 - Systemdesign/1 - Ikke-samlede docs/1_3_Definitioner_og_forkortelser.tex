\subsection{Definitioner og forkortelser}

\begin{itemize}
	\item GUI - Graphical User Interface.
	\item UI - User Interface.
	\item WPF - Windows Presentation Foundation.
	\item MVC - Model - View - Controller.
	\item API - Application programming interface.
	\item XML - Extensible Markup Language.
	\item SOAP - Simple Object Access Protocol.
	\item JSON - JavaScript Object Notation.
	\item AJAX - Asynchronous JavaScript and XML.
	\item GPS - Global Positioning System.
	\item UML - Unified Modeling Language.
	\item OO - Objektorienteret.
	\item ER - Entity relationship.
	\item CRUD - Create, read, update and delete.
	\item URI - Uniform resource identifier.
	\item Simpel rute - En busrute som spænder to endestationer.
	\item Kompleks rute - En busrute som spænder mere end to endestationer.
\end{itemize}
