\subsection{Implementeringssprog og værktøjer}
Herunder beskrives udviklingsværktøjer. Afsnittet er ikke ment som et udtømmende afsnit, og derfor vil alle værktøjer der har været brugt under projektet, ikke nødvendigvis beskrives.

\subsubsection{C\#}
Hjemmesidens controllers samt simulatoren er skrevet i C\#. Dette er gjort, da begge gør brug af .NET frameworket og den bibloteker.
\subsubsection{Java}
Mobil-applikation er skrevet i Java, som er standard for android appliaktioner
\subsubsection{HTMl, CSS og JavaScript}
Selve viewet på adminstrator værktøjet, er bygget op som en standard hjemmeside. Det vil sige med HTML, CSS og JavaScipt. 
\subsubsection{MySQL}
Den distruberede database er baseret på MySQL, og derfor er oprettelse, funktioner og procedurer skrevet i dette sprog.
\subsubsection{Microsoft Visual Studio 2012}
Administrator delen og simulatoren er bygget på .NET frameworket, og derfor skrevet i Visual Studio.
\subsubsection{Eclipse}
Mobil-applikationen er skrevet til Android, og det værktøj som projektgruppen havde mest erfaring med, var Eclipse.
\subsubsection{MySQL Workbench} 
Dette program er brugt til at opsætte og ændre MySQL databasen.

\subsubsection{Microsoft Visio 2013}
Samtlige diagrammer i dette dokument er udbygget ved hjælp af Visio. Der er nogle enkelte diagrammer i kravspecifikationen der er blevet ved hjælp af online-værktøjet \url{creately.com}
\subsubsection{GIT}
Til versions styring er der blevet gjort brug af GIT, og commandline værktøjet GIT shell.
\subsubsection{TexMaker}
Alt dokumentation er skrevet i \LaTeX og som editerings-værktøj er der brugt TexMaker.
