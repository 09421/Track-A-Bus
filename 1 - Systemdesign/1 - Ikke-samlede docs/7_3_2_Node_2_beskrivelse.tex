\subsubsection{Node 2 beskrivelse - Webserver}
Denne enhed hoster administrations hjemmesiden\footnote{\url{www.trackabus.dk}} samt web servicen, som ligger på serveren nt21.unoeuro.com. Hjemmesiden er implementeret ved brug af ASP.NET og Web servicen er en ASP.NET webservice.
