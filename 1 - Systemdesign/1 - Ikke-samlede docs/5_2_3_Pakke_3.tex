\subsubsection{Pakke 3 - Data tilgangs laget}
Denne pakkes eneste formål er, at tilgå databasen når det bliver nødvendigt i logik laget. Dette lag kan enten læse eller skrive til database, alt efter behov. Hentet data bliver ikke manipuleret, men returneret direkte til præsentations laget. Dette lag kan aldrig tilgås fra, eller returnere til, præsentations laget.
